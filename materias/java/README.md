# Java

Esta pasta contém a entrega da disciplina de **Java**, desenvolvida para o projeto **AETHER** na **Global Solution 2026**.

## Sobre a entrega

Nesta etapa, o foco foi criar uma implementação inicial do sistema em Java, aplicando conceitos de **Programação Orientada a Objetos** para representar parte da lógica do AETHER.

Além do código-fonte, a pasta também contém um PDF de documentação explicando a estrutura criada, as classes utilizadas e a relação da implementação com a proposta geral do projeto.

## Conteúdo

* Código-fonte em Java;
* Documentação da implementação;
* Estrutura inicial de classes;
* Aplicação de conceitos de POO;
* Organização de atributos e métodos;
* Representação dos modos do AETHER;
* Testes iniciais da lógica do sistema;
* Base para futuras melhorias do protótipo.

## Status

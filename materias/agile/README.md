# Agile & Squads

Esta pasta contém a entrega da disciplina de **Agile & Squads**, desenvolvida para o projeto **AETHER** na **Global Solution 2026**.

## Sobre a entrega

Nesta etapa, o foco foi organizar o projeto de forma ágil, definindo a estrutura da equipe, as responsabilidades, o planejamento das atividades e o backlog inicial da solução.

A documentação apresenta como o desenvolvimento do AETHER foi planejado e dividido entre os integrantes da squad.

## Conteúdo

* Organização da equipe;
* Definição de papéis;
* Divisão de responsabilidades;
* Planejamento das entregas;
* Backlog do projeto;
* Priorização das funcionalidades.

# Database Design

Esta pasta contém a entrega da disciplina de **Database Design**, desenvolvida para o projeto **AETHER** na **Global Solution 2026**.

## Sobre a entrega

Nesta etapa, o foco foi estruturar a modelagem do banco de dados do AETHER, organizando as informações necessárias para o funcionamento do sistema.

A documentação apresenta as entidades, atributos e relacionamentos pensados para armazenar dados sobre ambientes, sensores, alertas, recursos, eventos críticos e modos de operação.

## Conteúdo

* Modelagem conceitual;
* Definição das entidades;
* Atributos principais;
* Relacionamentos entre tabelas;
* Organização dos dados do sistema;
* Estrutura para registro de sensores, recursos e alertas;
* Base para futuras consultas e análises.
